2.ps aux ->lookup proccess info

4.dpkg -l->ubuntu installed package
5.mkdir name--->make a dir

8.mv yan.txt mi.txt-->rename a file

10.ln -s blah2 blah2-soft-->make soft link
11.rm -r mydir--->delete a dir
12.cp /dir/y.txt /dir3/-->copy a file to dir
13.service iptables stop-->turn off firewall
14.chkconfig iptables off->turn off firewall permanentlly
